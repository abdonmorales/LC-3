# lc3tools

> LC3 tools frontend

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn dev

# build electron application for production
yarn build

# build the macOS universal package explicitly
yarn build:mac:universal

```

This project was originally generated with [electron-vue](https://github.com/SimulatedGREG/electron-vue) and now builds with electron-vite.
